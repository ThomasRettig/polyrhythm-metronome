## Polyrhythm Metronome

Polyphonic metronome with geometric visualization. Created with React, Redux, Two.js, GSAP, and Bulma.

![screencast](https://media.giphy.com/media/fUqqpOXqsR4abYw9Ay/giphy.gif)
